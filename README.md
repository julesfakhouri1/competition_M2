# Enchanted Tools — Espace Admin

Interface d'administration pour gérer les modules de l'espace démo Enchanted Tools.

## Stack

- **Next.js 16** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Supabase** — base de données + authentification + stockage

## Fonctionnalités

- Authentification sécurisée (email/mot de passe)
- Création et configuration des blocs de modules (numéro, nom, description, audio/vidéo, images)
- Placement des modules en drag & drop sur le plan de l'espace

## Installation

```bash
npm install
```

Créer un fichier `.env.local` à la racine :


```bash
npm run dev
```

Ouvrir [http://localhost:3000/
